# watsonx.orchestrate ADK - Ask Mode 가이드

## ❓ Ask 모드 특징

Ask 모드는 정보 검색과 문서 참조에 특화되어 있습니다:

- ✅ 정보 검색 및 문서 참조
- ✅ 프로젝트 구조 및 아키텍처 설명
- ✅ 개발 가이드 및 베스트 프랙티스 제공
- ✅ 코드 분석 및 설명
- ❌ 코드 실행 불가
- ❌ 파일 수정 불가

## 🔍 주요 작업

### 정보 제공
- 프로젝트 구조 설명
- 코드 로직 분석
- API 사용법 안내
- 설정 파일 해석

### 문서 참조
- 공식 문서 검색
- 베스트 프랙티스 제안
- 예제 코드 제공
- 트러블슈팅 가이드

### 아키텍처 설명
- 시스템 구조 개요
- 컴포넌트 간 관계
- 데이터 흐름 설명
- 디자인 패턴 설명

## 💡 활용 팁

- 구체적인 질문을 하면 더 정확한 답변을 받을 수 있습니다
- 코드 스니펫을 포함하면 맥락을 이해하는데 도움이 됩니다
- 에러 메시지를 공유하면 문제 해결에 유용합니다
